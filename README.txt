1. 트래킹 하고 싶은 지역의 아파트 정보를 가지고 있는 엑셀 실행.

2. A ~ AL까지 복사 한 후 엑셀 시세표 엑셀 파일에서 "정보입력" sheet 에 복사 붙여넣기

3. 엑셀 시세표에서 면적입력 버튼 클릭.(여러 동에 대한 sheet들이 생성됨.)

4. 다른 이름으로 저장하기를 하고 GetData.py와 같은 경로에 놓기.(현재는 "시세표 지역명"으로 만들어놓음)

5. keyword.json 파일에 

## ~ ## 사이의 text는 수정해야함.

{"지역명" : "##지역이름##", "파일경로":"##./엑셀 파일이름.xlsm##"}

포맷에 맞춰서 추가하기.(트래킹하고 싶은 지역만 추가해놓으면 됨)

6. GetData.py 실행.

# 검색 지역 이름은 "구"가 있는 도시는 구별로 검색이 가능하고, "구"가 없는 도시는 해당 도시 이름으로 검새갛면 됨.
ex) 
군산시 : "구"가 없음. 군산시로 검색 하면됨.
원주시 : "구"가 없음. 원주시로 검색하면 됨.
전주시 : "완산구"가 있음. 구 별로 입력해줘야함.
서울시 : "송파구", "강남구", "서초구" 등 여러 구가 있음. 구 별로 입력 해줘야함.
